# uuid-nanoid

[![npm version](https://img.shields.io/npm/v/uuid-nanoid.svg)](https://www.npmjs.com/package/uuid-nanoid)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Generate UUIDs and short unique IDs (no dependencies)

## Installation

```bash
npm install uuid-nanoid
```

## Functions

- `uuid` — Generate a v4 UUID
- `nanoid` — Generate a short unique ID (default 21 chars)
- `cuid` — Generate a collision-resistant ID
