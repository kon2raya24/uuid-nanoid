# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in uuid-nanoid, please report it responsibly.

**Do NOT open a public GitHub issue.**

Instead, email: turayalemmuel@gmail.com

## Response Time

- Acknowledgment within 48 hours
- Fix within 7 days for critical issues

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | ✅        |
